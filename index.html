About This Project

Judge This Book utilizes a convolutional neural net (CNN) in order to determine which of 30 genres a book cover is most likely to belong in. It was trained using data from the UchidaLab dataset, containing 57,000 covers evenly split into 30 genres. Using the test set of 5,700 covers, evenly split into 30 genres, we were able to achieve a ~26% validation accuracy.
